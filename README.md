****Overview****

This project presents an Online Retail Store Analysis Dashboard built to uncover insights into sales performance, customer behavior, and product distribution across different countries.
The dashboard visualizes key metrics such as revenue trends, quantity sold, average unit price, and customer contribution using interactive charts and maps.

The goal of this project is to help business teams make data-driven decisions by exploring patterns in retail transactions.
